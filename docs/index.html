Image2Excel Introduction

Overview

Image2Excel is a software tool designed to automate the process of finding and organizing product images, exporting the results to an Excel file. Currently, it runs exclusively on Windows and has not been tested on Linux or other platforms.

Key Features

Image2Excel provides the following functionality:

  1. Searches for images (png/jpg/jpeg) based on product codes and user-defined suffixes (e.g., product_code-01.jpg) in the input directory, copying matches to an Image2Excel_Export folder.
  2. Generates an Excel file (output_date_time.xlsx) with product codes and corresponding images, starting from cell A2.

Inputs

  • Directory containing images (supports local or SMB Shared).
  • Input file (txt/xlsx, supports local or SMB Shared) with a list of product codes, one per line.
  • Up to four user-defined search suffixes (e.g., 01, 06).

Outputs

  • Image2Excel_Export directory containing images matched to product codes.
  • Excel file with columns for product codes and their corresponding images.

Example Excel output layout:

A B C
abc0123abc012-01.jpgabc012-06.jpg
abc456abc456-01.jpgabc456-06.jpg

Step-by-Step User Guide

Follow these steps to use Image2Excel effectively:

  1. Launch the Application:
    • Run Image2Excel.exe on a Windows system.
    • The GUI will load with previously saved settings from %APPDATA%\Image2Excel\settings.json.
  2. Select Input Files and Folders:
    • Product List: Click the "…" button next to "Product list" to select a .txt or .xlsx file containing product codes (one per line). Alternatively, use "File" > "Open Product List" from the menu.
    • Image Folder: Click the "…" button next to "Image folder" to choose a directory containing images (png/jpg/jpeg). You can also use "File" > "Select Image Folder."
    • Output Folder: Click the "…" button next to "Output folder" to specify where the Image2Excel_Export folder and Excel file will be saved. Use "File" > "Select Output Folder" as an alternative.
  3. Enter Image Suffixes:
    • Input up to four suffixes (e.g., 01, 06) in the "Image Suffixes" fields.
    • These suffixes are used to match images (e.g., product_code-01.jpg). Empty fields are ignored.
  4. Run the Process:
    • Click the "Run" button or press Ctrl+R.
    • The application will:
      • Validate inputs, ensuring all paths are provided and the output folder is not inside the image folder.
      • Search for images matching each product code with the specified suffixes.
      • Copy matched images to the Image2Excel_Export folder.
      • Create an Excel file (output_date_time.xlsx) with product codes and their images.
      • Display progress in a progress bar and log status (OK, Missing Images, or Errors) in the log table.
  5. Manage the Process:
    • Pause: Click "Pause" or press Ctrl+P to temporarily halt processing.
    • Resume: Click "Resume" or press Ctrl+O to continue.
    • Stop: Click "Stop" or press Ctrl+S to terminate the process.
  6. Filter Logs:
    • Use the "Filter" dropdown to view specific log entries: "All," "OK," "Missing Images," or "Errors."
  7. Save Settings:
    • Input paths and suffixes are automatically saved to %APPDATA%\Image2Excel\settings.json when modified or when the process is run.
  8. View Output:
    • Use "File" > "Open Export Folder" to open the Image2Excel_Export folder containing matched images and the Excel file.
  9. Switch Themes:
    • Navigate to "View" > "Light Mode" or "Dark Mode" to toggle between themes.
  10. Access Help:
    • Click "Help" > "About" to view application details, including version and developer information.
